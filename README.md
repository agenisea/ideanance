# Ideanance

**Governance is a design primitive.**

Ideanance is an open-source design workspace where AI governance policies are wired into agent design from day one — not bolted on after.

- **Activate** governance frameworks (NIST AI RMF, EU AI Act, custom)
- **Generate** evaluation criteria from governance policies
- **Export** structured handoff packages with governance provenance
